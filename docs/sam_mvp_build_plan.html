Day 2
MonoGraph — the AI brain
Static imports + git log co-change. The intelligence that makes sparse checkout feel smart.
AST
Static import analyzer
Python
tree-sitter parses JS/TS/Python/Java/Go/C#. Outputs dependency graph. Runs once on clone, caches to .sam/graph.json.
Supports polyglot monorepos out of the box
GIT
Co-change miner
Python
Mines git log for file co-change frequency. "These two modules changed together 87 times — you need both." Scored 0.0–1.0.
The signal static analysis always misses
